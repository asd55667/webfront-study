# 盒模型

盒子主要的 5 个属性：width、height、padding、border、margin

标准盒子模型：
宽度=内容的宽度（content）+ border + padding + margin
低版本 IE 盒子模型：
宽度=内容宽度（content+border+padding）+ margin

**box-sizing** 用来控制元素的盒子模型的解析模式，默认为 content-box
