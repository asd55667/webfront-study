# 清除浮动

浮动元素碰到包含它的边框或者浮动元素的边框
浮动元素会漂浮在文档流的块框上

> 造成的问题

- 父元素的高度无法被撑开，影响与父元素同级的元素
- 与浮动元素同级的非浮动元素（内联元素）会跟随其后
- 若非第一个元素浮动，则该元素之前的元素也需要浮动

清除方式:

- 触发自身或父级 BFC
- 父级 div 定义 height
- 最后一个浮动元素后加空 div 标签 并添加样式 clear:both。
- 父级 div 定义 zoom
- :after 伪类 作用于父类浮动元素
